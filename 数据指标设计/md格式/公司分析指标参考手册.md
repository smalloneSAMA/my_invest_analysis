# 📊 公司分析指标参考手册

本手册汇总了A股上市公司分析中常用的原子数据和组合数据指标，按功能分类整理，方便快速查阅。

---

### 1. 公司基本信息（公司身份标签）

| 指标名称   | 说明                               |
| ---------- | ---------------------------------- |
| 证券代码   | 股票在交易所的唯一代码             |
| 证券简称   | 股票的简称名称                     |
| 公司全称   | 上市公司的法定全称                 |
| 上市日期   | 公司在交易所上市交易的日期         |
| 所属行业   | 公司所属的行业分类（常用申万一级） |
| 上市板块   | 主板/科创板/创业板/北交所等        |
| 注册地址   | 公司工商注册所在地                 |
| 办公地址   | 公司实际办公所在地                 |
| 法定代表人 | 公司法定的法定代表人姓名           |
| 主营业务   | 公司主要经营业务范围               |
| 员工总数   | 公司全部员工人数                   |

### 2. 股本与股份数据（股本规模与结构）

| 指标名称 | 说明                               |
| -------- | ---------------------------------- |
| 总股本   | 公司已发行的全部股份数量           |
| 流通股本 | 已发行普通股中非限售的股份数量     |
| 限售股本 | 已发行普通股中尚在限售期的股份数量 |
| A股股本  | 在境内发行的人民币普通股数量       |
| B股股本  | 在境内发行的外币普通股数量         |
| H股股本  | 在香港发行的境外上市外资股数量     |
| 股东总数 | 报告期末登记在册的股东人数         |

| ---------- | ------------------------------- |

| ST状态     | 公司是否被实施特别处理（是/否） |
| 停牌标记   | 公司股票当日是否停牌（是/否）   |
| 上市天数   | 公司自上市之日起的累计天数      |

| 应收账款               | 已销售未收回的货款       |
| 存货                   | 库存商品、原材料等       |
| 固定资产               | 房屋、机器设备等长期资产 |
| 总负债                 | 公司全部负债合计         |
| 流动负债               | 一年内到期的负债合计     |
| 非流动负债             | 一年以上到期的负债合计   |
| 短期借款               | 一年内到期的银行借款     |
| 长期借款               | 一年以上到期的银行借款   |
| 应付账款               | 已采购未支付的货款       |
| 所有者权益（股东权益） | 总资产 - 总负债          |
| 归属于母公司股东权益   | 母公司股东应占的权益部分 |
| 资本公积               | 股本溢价等资本性公积     |
| 盈余公积               | 从净利润中提取的公积     |
| 未分配利润             | 累计未分配的净利润       |

### 5. 利润表数据（盈利规模）

| 指标名称                 | 说明                                    |
| ------------------------ | --------------------------------------- |
| 营业总收入               | 公司主营业务收入合计                    |
| 营业收入                 | 销售商品或提供服务所得收入              |
| 营业成本                 | 与营业收入对应的成本                    |
| 营业利润                 | 营业收入 - 营业成本 - 期间费用等        |
| 利润总额                 | 营业利润 + 营业外收支                   |
| 净利润                   | 利润总额 - 所得税                       |
| 归属于母公司股东净利润   | 母公司股东应占的净利润部分              |
| 扣除非经常性损益后净利润 | 扣除一次性损益后的净利润                |
| 毛利润                   | 营业收入 - 营业成本（可直接从报表获取） |
| 经营活动净收益           | 经营活动产生的收益                      |
| 公允价值变动净收益       | 金融资产公允价值变动收益                |
| 利息费用                 | 借款利息支出                            |
| 基本每股收益（披露值）   | 财报直接披露的每股收益                  |
| 稀释每股收益（披露值）   | 考虑潜在稀释因素后的每股收益            |

### 7. 估值指标（判断贵贱）

| 指标名称             | 计算公式                                | 说明                         |
| -------------------- | --------------------------------------- | ---------------------------- |
| 总市值               | 收盘价 × 总股本                        | 市场给予公司的总定价         |
| 流通市值             | 收盘价 × 流通股本                      | 实际可自由交易的市值         |
| 市盈率（PE）         | 总市值 ÷ 归属于母公司股东净利润        | 静态估值，衡量回本年限       |
| 滚动市盈率（PE-TTM） | 总市值 ÷ 近12个月归母净利润            | 剔除季节因素，更及时         |
| 市净率（PB）         | 总市值 ÷ 归属于母公司股东权益          | 股价相对净资产的溢价         |
| 市销率（PS）         | 总市值 ÷ 营业总收入                    | 适用于亏损或早期成长公司     |
| 市现率（PCF）        | 总市值 ÷ 经营活动现金流净额            | 现金流视角的估值             |
| PEG                  | PE-TTM ÷ （归母净利润同比增速 × 100） | 衡量估值与增速匹配度，<1低估 |
| 股息率               | 近一年现金分红总额 ÷ 总市值            | 现金分红收益率               |
| EV/EBITDA            | 企业价值（EV） ÷ 息税折旧摊销前利润    | 排除折旧和资本结构差异的估值 |

---

### 8. 盈利能力指标（判断赚钱效率）

| 指标名称               | 计算公式                                | 说明                                   |
| ---------------------- | --------------------------------------- | -------------------------------------- |
| 净资产收益率（ROE）    | 归母净利润 ÷ 归母股东权益（加权平均）  | 股东资本赚钱效率，长期>15%优秀         |
| 总资产收益率（ROA）    | 净利润 ÷ 总资产                        | 总资产赚钱效率                         |
| 投入资本回报率（ROIC） | 税后经营净利润 ÷ （总资产 - 流动负债） | 剔除杠杆干扰的核心业务回报             |
| 毛利率                 | （营收 - 营业成本） ÷ 营收             | 产品定价权和护城河                     |
| 净利率                 | 净利润 ÷ 营收                          | 每元收入最终留下多少纯利               |
| 营业利润率             | 营业利润 ÷ 营收                        | 主营业务的盈利能力（不含非经常性损益） |
| 扣非净利率             | 扣非净利润 ÷ 营收                      | 剔除一次性损益后的真实净利率           |
| 基本每股收益（计算值） | 归母净利润 ÷ 总股本（加权平均）        | 每股享有的利润，与披露值可能略有差异   |

---

### 9. 营运能力指标（判断周转效率）

| 指标名称         | 计算公式                   | 说明                       |
| ---------------- | -------------------------- | -------------------------- |
| 总资产周转率     | 营业总收入 ÷ 平均总资产   | 总资产产生收入的效率       |
| 应收账款周转率   | 营业总收入 ÷ 平均应收账款 | 回收账款的速度             |
| 存货周转率       | 营业成本 ÷ 平均存货       | 存货变现速度               |
| 固定资产周转率   | 营业总收入 ÷ 平均固定资产 | 重资产的利用效率           |
| 应收账款周转天数 | 365 ÷ 应收账款周转率      | 从赊销到收回现金的平均天数 |
| 存货周转天数     | 365 ÷ 存货周转率          | 从进货到卖出的平均天数     |

---

### 10. 偿债与资本结构（判断财务风险）

| 指标名称     | 计算公式                                 | 说明                                   |
| ------------ | ---------------------------------------- | -------------------------------------- |
| 资产负债率   | 总负债 ÷ 总资产                         | 杠杆水平，>70%需警惕                   |
| 权益乘数     | 总资产 ÷ 归母股东权益                   | 杠杆放大倍数                           |
| 流动比率     | 流动资产 ÷ 流动负债                     | 短期偿债安全垫，>1较安全               |
| 速动比率     | （流动资产 - 存货） ÷ 流动负债          | 更保守的短期偿债能力，>1为佳           |
| 利息保障倍数 | 息税前利润（EBIT） ÷ 利息费用           | 经营利润覆盖利息的倍数，<3倍警示       |
| 带息负债比率 | （短期借款+长期借款+应付债券） ÷ 总负债 | 有息刚性债务占比，越高财务费用压力越大 |

---

### 11. 成长性指标（判断增长速度）

| 指标名称             | 计算公式                                                | 说明                       |
| -------------------- | ------------------------------------------------------- | -------------------------- |
| 营收同比增长率       | （本期营收 - 上年同期营收） ÷ 上年同期营收             | 业务规模扩张速度           |
| 归母净利润同比增长率 | （本期归母净利 - 上年同期归母净利） ÷ 上年同期归母净利 | 盈利增长速度               |
| 扣非净利同比增长率   | （本期扣非净利 - 上年同期扣非净利） ÷ 上年同期扣非净利 | 剔除一次性损益后的真实增速 |
| 净资产增长率         | （本期归母权益 - 上期归母权益） ÷ 上期归母权益         | 净资产积累速度             |
| 单季净利润环比增长率 | （本季净利 - 上季净利） ÷ 上季净利                     | 剔除基数效应的景气趋势指标 |

---

### 12. 现金流质量指标（判断利润含金量）

| 指标名称                 | 计算公式                   | 健康标准            | 说明                           |
| ------------------------ | -------------------------- | ------------------- | ------------------------------ |
| 净现比（净利润现金含量） | 经营现金流净额 ÷ 净利润   | >1 优秀；<0.5 警惕  | 利润是否真金白银               |
| 收现比（销售现金比率）   | 经营现金流净额 ÷ 营业收入 | >0.8 合格；>1 优秀  | 每元营收收回的现金             |
| 自由现金流（FCF）        | 经营现金流净额 - 资本支出  | 正数且持续增长      | 真正可自由支配的现金           |
| FCF/营业收入             | 自由现金流 ÷ 营业收入     | ≥5% 堪称“印钞机” | 营收中的真实现金含量           |
| 资本支出/经营现金流      | 资本支出 ÷ 经营现金流净额 | 长期>100% 警示      | 赚的钱不够再投资，依赖外部融资 |

---

### 13. 财务质量与排雷指标（识别暴雷风险）

| 指标名称                 | 计算公式                     | 风险警示线                 | 说明                           |
| ------------------------ | ---------------------------- | -------------------------- | ------------------------------ |
| 商誉/净资产              | 商誉 ÷ 归母股东权益         | >30% 高度警惕              | 商誉减值风险，A股最大暗雷      |
| 应收账款/营业收入（TTM） | 应收账款 ÷ 近12个月营收     | 显著高于同行均值           | 利润可能是“白条”，有造假嫌疑 |
| 存货/总资产              | 存货 ÷ 总资产               | 激增且周转放缓             | 跌价风险，尤其电子/服装行业    |
| 资产减值损失/净利润      | 当期资产减值损失 ÷ 净利润   | 占比过大                   | 资产质量存疑，利润含金量低     |
| 审计意见类型             | 标准/保留/否定/无法表示      | **非标准意见即回避** | 最强警示信号                   |
| 大股东股权质押比例       | 大股东质押股数 ÷ 其持股总数 | >60% 亮红灯                | 强制平仓风险，可能引发暴跌     |

### 14. 股东回报与资金面（衡量股东回报）

| 指标名称           | 计算公式                                       | 解读                                      |
| ------------------ | ---------------------------------------------- | ----------------------------------------- |
| 分红融资比         | 累计现金分红总额 ÷ 累计融资总额（含IPO+增发） | >1 表示公司是“反哺型”资产，长期价值核心 |
| 股息支付率         | 现金分红总额 ÷ 归母净利润                     | 赚100元分多少，30%-70%合理                |
| 回购注销金额       | 公司回购并注销的股票市值                       | 变相分红，提升每股价值                    |
| 综合股东回报率     | （分红总额 + 回购注销金额） ÷ 总市值          | 比股息率更全面衡量股东回报                |
| 产业资本净增减持额 | 增持金额 - 减持金额（统计期）                  | 持续净减持是强烈负面信号                  |
| 机构持仓占比       | 机构持股数 ÷ 流通股本                         | 机构增持代表专业认可，但过高也拥挤        |

---

### 15. 市场情绪与资金面（感知温度，无计算公式）

| 指标名称         | 说明                                                 | 使用场景                                   |
| ---------------- | ---------------------------------------------------- | ------------------------------------------ |
| 换手率           | 成交量 ÷ 流通股本                                   | 判断活跃度和筹码交换程度，过高说明短期过热 |
| 涨跌幅           | （当日收盘价 - 前日收盘价）÷ 前日收盘价             | 衡量短期价格变动幅度                       |
| 振幅             | （当日最高价 - 当日最低价）÷ 前日收盘价             | 反映日内波动剧烈程度                       |
| 量比             | 当日每分钟均量 ÷ 过去5日每分钟均量                  | 放量（>1.5）或缩量信号                     |
| 市净率历史分位点 | （当前PB - 历史最低PB）÷（历史最高PB - 历史最低PB） | 判断估值在历史水位，>80%为高估区           |
| A股风险溢价      | （1/PE） - 10年期国债收益率                          | 数值越高，股票相对债券性价比越高           |
| 融资余额规模     | 每日融资买入额 - 偿还额（累计）                      | 杠杆资金做多意愿，激增代表短期过热         |
| 限售股解禁市值   | 未来特定时段解禁的市值规模                           | 解禁高峰前股价承压，需提前规避             |

---

### 16. 其他重要补充指标（仅说明）

| 指标名称     | 说明                                                 |
| ------------ | ---------------------------------------------------- |
| 预期差       | 实际业绩 vs 机构一致预期，超预期是上涨动力           |
| 管理层展望   | 业绩发布会、调研纪要中的前瞻信息，比财报数字更有价值 |
| 机构调研次数 | 反映市场关注度，次数激增往往预示潜在变化             |
