# 📊 原子数据指标清单_时间序列优化版

> 本手册汇总所有原子数据指标，以时间序列方式组织。第一个Sheet为字段字典（指标名称+说明），其余Sheet以“日期”为第一列，各指标按列排列。
> 使用说明：按日期逐行填入数据，可记录单家公司的时间序列，也可在“日期”后增加“证券代码”列扩展为多公司对比。

---

## 公司基本信息

| 指标名称     | 说明                        |
| ------------ | --------------------------- |
| 证券代码     | 股票在交易所的唯一代码      |
| 证券简称     | 股票的简称名称              |
| 公司全称     | 上市公司的法定全称          |
| 上市日期     | 公司在交易所上市交易的日期  |
| 所属行业     | 公司所属的行业分类          |
| 上市板块     | 主板/科创板/创业板/北交所等 |
| 公司注册地址 | 公司工商注册所在地          |
| 办公地址     | 公司实际办公所在地          |
| 法定代表人   | 公司法定的法定代表人姓名    |
| 主营业务     | 公司主要经营业务范围        |
| 员工总数     | 公司全部员工人数            |
