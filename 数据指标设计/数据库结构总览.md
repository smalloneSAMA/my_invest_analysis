📊 数据库设计总览

```Shell
   investment_analysis                                                                                                                                       
   ├── 元数据层 (2表)                                                                                                                                        
   │   ├── data_sources            ← 3个数据源已注册 (mootdx/tencent/eastmoney)                                                                              
   │   └── data_source_priority    ← 27条优先级规则 (9类别 × 3源)                                                                                            
   │                                                                                                                                                         
   ├── 主数据层 (1表)                                                                                                                                        
   │   └── companies               ← 公司基本信息                                                                                                            
   │                                                                                                                                                         
   ├── 原始数据层 (8表)                                                                                                                                      
   │   ├── daily_quotes            ← 日行情 + 复权因子 (坑1核心)                                                                                             
   │   ├── adjustment_events       ← 除权除息事件审计 (坑1配套)                                                                                              
   │   ├── balance_sheet           ← 资产负债表宽表                                                                                                          
   │   ├── income_statement        ← 利润表宽表                                                                                                              
   │   ├── cash_flow_statement     ← 现金流量表宽表                                                                                                          
   │   ├── share_capital           ← 股本历史                                                                                                                
   │   ├── dividend_records        ← 分红记录                                                                                                                
   │   └── audit_opinions          ← 审计意见                                                                                                                
   │                                                                                                                                                         
   ├── 衍生数据层 (3表) - 物化存储                                                                                                                           
   │   ├── derived_metrics_daily   ← 估值 + 市场情绪 (PE/PB/PS/PEG/股息率等)                                                                                 
   │   ├── derived_metrics_financial ← 盈利/营运/偿债/现金流质量/排雷 (ROE/ROA等)                                                                            
   │   └── derived_metrics_growth  ← 成长性 (同比/环比增长率)                                                                                                
   │                                                                                                                                                         
   └── 业务层设施 (2函数 + 3视图)                                                                                                                            
       ├── safe_divide()           ← NULL/0 安全除法 (坑3核心)                                                                                               
       ├── v_daily_quotes_priority ← 多源自动选TOP1 (坑2核心)                                                                                                
       ├── v_balance_sheet_priority← 财报多源TOP1                                                                                                            
       ├── v_daily_quotes_adjusted ← 自动前复权行情 (坑1核心)                                                                                                
       └── get_trading_days_before() ← 辅助函数
```
